Parámetros De Configuración SW (Service Worker) y Manifest

En este apartado de muestra cuales fueron los parámetros y el proceso de configuración de nuestro entorno para implemtar el Service Worker y el Manifest.

  1. Se creo este código está escrito en JavaScript y se utiliza en el contexto de un Service Worker, que es un script que se ejecuta en segundo plano en el navegador y permite realizar tareas como el control de caché, notificaciones push y otras funciones relacionadas con la mejora del rendimiento y la experiencia del usuario.

  1. Este script carga script.js de forma asíncrona. La palabra clave async indica que la descarga y ejecución del script no bloquearán la carga de la página. Este script puede contener funciones o lógica específica de la página.

  1. Este es un archivo de manifiesto de aplicación web progresiva (PWA) en formato JSON. El archivo manifest.json proporciona información sobre la aplicación web, como su nombre, iconos, colores temáticos y otras configuraciones importantes para mejorar la experiencia del usuario. A continuación, se explica cada propiedad del manifiesto:
  2. name: El nombre de la aplicación web. En este caso, es "Aplicaciones Web Progresivas - Producto 2"
    short_name: Un nombre más corto o abreviado para la aplicación. En este caso, es "PWA-P2"
    start_url: La URL de inicio de la aplicación web. En este caso, es "./index.html", lo que sugiere que la aplicación web debería comenzar en el archivo index.html en el directorio raíz
    display: Especifica cómo se debe mostrar la aplicación web. En este caso, "standalone" indica que la aplicación debe abrirse en su propia ventana, sin una barra de navegación
    theme_color: Define el color de fondo de la barra de estado (si es aplicable) y el color de fondo de la pantalla de inicio cuando la aplicación está en modo "standalone". En este caso, es "#87C4A2"
    background_color: Define el color de fondo de la aplicación web. En este caso, es "#87C4A2"
    icons: Una lista de iconos que se utilizarán para representar la aplicación en diferentes contextos
    src: La ruta del archivo de imagen del icono
    sizes: El tamaño del icono
    type: El tipo de archivo de imagen (en este caso, "image/png")

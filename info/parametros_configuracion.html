Parámetros De Configuración

En este apartado de muestra cuales fueron los parámetros y el proceso de configuración de nuestro entorno.

  1. Después de instalar XAMPP, abre el panel de control de XAMPP.
  2. Inicia el módulo Apache haciendo clic en los botones correspondientes.
  3. Asegúrate de que Apache esté escuchando en el puerto 80 (el puerto web estándar).

  1. Crea una carpeta para tu proyecto web en el directorio "htdocs" de XAMPP. Por ejemplo, puedes crear una carpeta llamada "miweb" dentro de "htdocs".

  1. Dentro de tu carpeta de proyecto, crea un archivo "index.html" para la página principal.
  2. Agrega una carpeta donde alojarás los frameworks y bibliotecas que utilizarás para tu aplicación web.
  3. Agrega una carpeta donde alojes tus scripts o html que utilizarás para manejar las solicitudes de AJAX.
  4. Asegúrate de que Apache esté escuchando en el puerto 80 (el puerto web estándar).

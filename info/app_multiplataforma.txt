        <div class="col-md-6 mt-4">
            
            <h4>¿Qué es una aplicación multiplataforma?</h4>

            <p>
                Una aplicación multiplataforma es un tipo de software que está 
                diseñado para funcionar en múltiples sistemas operativos o plataformas 
                informáticas sin necesidad de ser reescrito o adaptado significativamente 
                para cada una de ellas. Estas aplicaciones son desarrolladas de manera 
                que puedan ejecutarse de manera nativa en diferentes sistemas operativos 
                y hardware sin problemas significativos.
            </p>

            <h4>
                Características 
                <i class="fa-solid fa-list-check me-2" style="color: #602a5d;"></i>
            </h4>

            <ul>
                <li>
                    <b>Compatibilidad con múltiples plataformas: </b>
                    La característica más destacada de una aplicación multiplataforma 
                    es su capacidad para funcionar en diferentes sistemas operativos y 
                    dispositivos sin modificaciones significativas en el código fuente.
                </li>
                <li>
                    <b>Código compartido: </b>
                    Estas aplicaciones permiten a los desarrolladores compartir 
                    una gran parte del código fuente entre las distintas versiones
                    de la aplicación para cada plataforma.
                </li>
                <li>
                    <b>Menor costo de desarrollo: </b>
                    Al escribir y mantener un único código base en lugar de 
                    múltiples versiones para cada plataforma, las aplicaciones 
                    multiplataforma a menudo resultan más económicas en términos 
                    de desarrollo y mantenimiento.
                </li>
            </ul>

Categoría Definición Características Ventajas Desventajas Herramientas de Desarrollo
Aplicación Web Una aplicación accesible a través de un navegador web sin necesidad de instalación en el dispositivo. - Acceso desde cualquier navegador. - Facilidad de acceso. - Requiere conexión a Internet. HTML, CSS, JavaScript
Aplicación Web Orientada a Servicios Similar a una aplicación web, pero interactúa con servicios web para acceder a datos y funcionalidades. - Integra servicios web. - Mayor funcionalidad que las web estáticas. - Dependencia de servicios externos. HTML, CSS, JavaScript, APIs
Aplicación Móvil Nativa Una aplicación diseñada y desarrollada específicamente para una plataforma móvil (iOS o Android). - Alto rendimiento y experiencia nativa. - Acceso a hardware y funciones del dispositivo. - Desarrollo separado para cada plataforma. Swift (iOS), Java/Kotlin (Android)
Aplicación Móvil Multiplataforma Una aplicación desarrollada una vez y compatible con múltiples plataformas (iOS y Android, por ejemplo). - Reutilización de código. - Menos esfuerzo de desarrollo. - Posible limitación en rendimiento y acceso. React Native, Flutter, Xamarin
Progressive Web App (PWA) Una aplicación web que ofrece una experiencia similar a la de una aplicación nativa, incluyendo offline. - Funciona offline. - Accesible sin descargas ni instalaciones. - Limitaciones en acceso a hardware. HTML, CSS, JavaScript, Service Workers

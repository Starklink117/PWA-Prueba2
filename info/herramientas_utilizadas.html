Herramietas utilizadas

En este apartado mostramos las herramientas que se utilizaron para desarrollar nuestra aplicación web.

Bootstrap

Bootstrap

Es un marco de trabajo o framework de CSS que ofrece componentes predefinidos y personalizables para crear interfaces web responsivas y adaptativas. Utiliza clases para aplicar estilos a los elementos HTML y también incluye plugins de JS para añadir funcionalidades como carruseles, pestañas, modales, etc.

Visual Code

Visual Code

Es un editor de código fuente desarrollado por Microsoft que ofrece soporte para múltiples lenguajes de programación, como HTML, CSS, JS y Bootstrap. Tiene un rico ecosistema de extensiones que amplían sus funcionalidades y facilitan el desarrollo web.

XAMPP

XAMPP

Es un paquete de software que contiene Apache, MySQL, PHP y Perl. Se utiliza para crear y probar aplicaciones web en un entorno local.

HTML

HTML

Es el lenguaje de marcado que define la estructura y el contenido de las páginas web. Utiliza etiquetas para crear elementos como encabezados, párrafos, listas, enlaces, imágenes, etc.

CSS

CSS

Es el lenguaje de estilo que define la apariencia y el diseño de las páginas web. Utiliza reglas para aplicar propiedades como color, fuente, tamaño, posición, etc., a los elementos HTML.

JS

JS

Es el lenguaje de programación que permite añadir interactividad y dinamismo a las páginas web. Utiliza scripts para manipular el DOM, realizar operaciones lógicas, validar datos, crear animaciones, etc.
